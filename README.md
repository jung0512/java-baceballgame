# 야구게임 
게임규칙
랜덤하게 생성된 숫자 3개를 유저가 입력한 숫자와 비교하여 <br>
자리수가 같은 숫자가 있으면 1스트라이크 숫자만 같으면 1볼로 3스트라이크가 되면 끝나는 게임이다<br>
